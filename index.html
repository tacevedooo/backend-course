Curso de Desarrollo Backend

Aprende a construir aplicaciones backend modernas desde cero hasta nivel profesional utilizando Node.js, Express y bases de datos reales.

¿Qué necesitas saber? (Prerrequisitos)

Fundamentos de Programación

Variables, tipos de datos, operadores, condicionales, bucles, funciones, arrays y objetos.

JavaScript Moderno (ES6+)

let y const, template strings, destructuring de objetos y arrays.

Terminal y Sistema Archivos

Navegación básica en la terminal y manejo de archivos y carpetas.

¿Qué aprenderás? (Contenido del curso)

1. Fundamentos del Backend

En este módulo aprenderás cómo funciona realmente la web detrás de escena. Entenderás conceptos clave como HTTP, APIs, JSON y la comunicación entre cliente y servidor, que son la base de todo sistema backend.

2. Node.js desde Cero

Aprenderás cómo usar Node.js para crear servidores reales con JavaScript. Descubrirás cómo funciona el event loop, los módulos de Node y construirás tu primer servidor desde cero.

3. Creación de APIs REST con Express

Durante este módulo aprenderás a desarrollar APIs REST profesionales utilizando Express. Crearás rutas, controladores y middlewares para construir aplicaciones backend modernas.

4. Bases de Datos (SQL y MongoDB)

Aprenderás cómo guardar, consultar y administrar datos en aplicaciones backend usando bases de datos relacionales (SQL) y NoSQL (MongoDB).

5. Autenticación y Seguridad

En este módulo aprenderás a proteger tus aplicaciones backend implementando autenticación con JWT, manejo seguro de contraseñas y protección contra ataques comunes.

6. Arquitectura Backend Profesional

Aprenderás cómo se estructuran los proyectos backend a nivel profesional, aplicando arquitectura, patrones de diseño, testing y despliegue de aplicaciones.
