Video de la Clase

Nuestro primer servidor web

A diferencia de PHP o Apache, en Node.js nosotros mismos levantamos el servidor usando el módulo http. Aquí tienes el código base:

server.js
const http = require('http');

const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('¡Hola, Backend desde Node.js!');
});

server.listen(3000, () => {
    console.log('Servidor corriendo en el puerto 3000');
});
